Test Custom Titles
==================

This page demonstrates the custom title functionality for UiO directives.

Using Custom Titles
-------------------

.. uio-note:: Husk

   This is a note with a custom title "Husk" instead of the default "Merk".

.. uio-do:: Prøv dette

   This is a tip with a custom title "Prøv dette" instead of the default "Tips".

.. uio-dont:: Pass på

   This is a warning with a custom title "Pass på" instead of the default "OBS!".

.. uio-exercise:: Øvelse 1

   This is an exercise with a custom title "Øvelse 1" instead of the default "Oppgave".

.. uio-question:: Spørsmål 1

   This is a question with a custom title.
